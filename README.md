# hydrocarbon-transportation
## About
Completed programme created within course 'Algorithmic Languages' during the autumn semester 2025. It is a console application primitively implementing some tasks from Oil&Gas industry, precisely from hydrocarbon (gas) transportation.
## Functionality
Console application provide creating gas transportation network. Some attributes of pipelines and compressor stations could be edited. There is a realizations of topologic sort and shortest path (Dijkstra) algorithms that could be applied to existing gas transportation network.
